Tổng quan về React Native

React Native là gì?

Khái niệm

  1. Khái niệm trên sách vở

    React-Native là một framework cho phép lập trình viên chỉ sử dụng một ngôn ngữ lập trình javascript xây dựng các ứng dụng gốc (trích http://reactnative.vn/lap-trinh-mobile-app-voi-react-native/)

  2. Thêm thắc mắc

    React là gì???

    Ứng dụng gốc là gì???

Giải quyết thắc mắc

  1. React là gì?

    là một thư viện JavaScript được xây dựng bởi một kỹ sư Facebook tên Jordan Walke

    • Triển khai trên newsfeed facebook năm 2011 và instagram.com năm 2012
    • Công bố lên Github năm 2013

  2. Ứng dụng gốc (Native app) là gì?

    các ứng dụng được viết riêng cho một nền tảng bằng ngôn ngữ riêng của nền tảng đó

    Ví dụ

    • Java trên Android
    • Objective-C trên iOS (giờ người ta sử dụng switf để lập trình iOS)

Ra đời từ sự thỏa mong muốn

Với mong muốn xây dựng ứng dụng chạy được trên nhiều nền tảng các nhà phát triển của facebook đã tạo ra React Native

  • Xây dựng đầu tiên trên iOS vào tháng 3 năm 2015

  • Cuối năm 2015 xây dựng trên Android

Lập trình với React Native

Cài đặt React Native

  1. Yêu cầu bắt buộc phải có

    • Lập trình Android:

      Đối với Linux hay Windows cần phải có Nodejs, React Native CLI và Android Studio

    • Lập trình iOS:

      Sử dụng hệ hành MacOS cần phải Nodejs, Watchman, React Native CLI và XCode

  2. Cách cài đặt

    Cài đặt React Native rất đơn giản

    Đọc và làm theo hướng dẫn tại https://facebook.github.io/react-native/docs/getting-started.html

Xây dựng và chạy chương trình

  • Xây dựng

    Ví dụ xây dựng một ứng dụng ExampleApp

    react-native init ExampleApp
cd ExampleApp
react-native start
react-native run-android

Xây dựng và chạy chương trình

  • Thiết bị chạy chương trình

    • Chạy bằng máy ảo

      • Sử dụng máy ảo trong Android Studio với ứng dụng Android
      • Sử dụng iOS Simulator từ XCode

    • Chạy bằng máy thật thông qua cổng USB, WIFI
