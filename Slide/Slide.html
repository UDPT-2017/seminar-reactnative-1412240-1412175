Tổng quan về React Native

React Native là gì?

Khái niệm

  1. Khái niệm trên sách vở

    React Native là một framework cho phép các lập trình viên xây dựng các ứng dụng Native mà chỉ sử dụng ngôn ngữ lập trình javascript. React Native cho phép bạn xây dựng các ứng dụng trên android và ios chỉ với một ngôn ngữ thống nhất là javascript nhưng mang lại trải nghiệm Native app thực sự.

Lập trình với React Native

Cài đặt React Native

Yêu cầu bắt buộc phải có

  • Lập trình Android:

    Nếu sử dụng hệ điều hành Linux hay Windows thì đều cần phải có Node.js, React Native CLI và Android Studio.

  • Lập trình iOS:

    Cần phải sử dụng hệ điều hành MacOS hoặc cần phải giả lập MacOS bởi vì Apple chỉ chơi với một mình nhà nó không chơi chung với nhà hàng xóm cần phải có Node.js, Watchman, React Native CLI và XCode

Cài đặt React Native

Hướng dẫn cài đặt

  • Trước hết bạn phải cài đặt Node.js nhé
  • Sau đó cài đặt React Native CLI bằng cách chạy câu lệnh trong Terminal npm install -g react-native-cli
  • Nếu bạn lập trình Android thì phải cài đặt thêm Android Studio. Nếu bạn lập trình iOS có thể cài thêm XCode nữa nhé

  • Đọc và làm theo hướng dẫn tại https://facebook.github.io/react-native/docs/getting-started.html

Xây dựng chương trình với React Native

  • Tạo React Native Project

    Chúng ta sử dụng React Native command line interface (React Native CLI) để tạo một React Native project nhé. Ví dụ ở đây mình tạo một project có tên là HelloWorldApp.

    react-native init HelloWorldApp
cd HelloWorldApp
